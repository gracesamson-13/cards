GRACE SAMSON:
Gin Rummy:

This is a spin off on gin rummy, where the goal was to
win by having a set (same numbered) cards

the options were to have a draw, discard, and knock button.

Draw and discard are available to you to select from at each turn,
you can draw cards to get 3 in a row, and you can discard them for neatness,
when you have 3 cards with the same number, click knock to win
